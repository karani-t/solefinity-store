## 🚀 How to Start (5 Minutes)

```bash
# 1. Install
npm install

# 2. Configure
cp .env.example .env.local
# Edit: NEXTAUTH_SECRET, DATABASE_URL

# 3. Database
npx prisma migrate dev

# 4. Run
npm run dev

# 5. Open
# http://localhost:3000
# Login: admin@solefinity.com / password123
```
